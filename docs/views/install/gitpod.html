Install Gitpod to personal repository

Install the Gitpod app to your Github account here.

Add to your github profile and choose to give full access to all your repositories or select a repository.

Click install.

Gitpod on NHSUK organisation

This is for members of NHSUK specifically.

  1. Configure Gitpod on the NHSUK organisation here
  2. Choose the repository within the organisation to install Gitpod
  3. If you are a repository admin this will automatically install
  4. Click install

If you're not an admin on the repository you wish to install Gitpod on you will need to request the installation, an admin can approve this in the project, under the applications section.

Gitpod on NHSUK organisation

Configure Gitpod on the NHSUK organisation here.

If you are a repository admin this will automatically install. If you're not an admin on the repository you wish to install Gitpod on you will need to request the installation, an admin can approve this in the project section under applications.
